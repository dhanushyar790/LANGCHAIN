# ⚡ LangChain Demo

A simple demonstration of LangChain modules working together — loaders, text splitters, embeddings, FAISS vector store, and OpenAI chat interface. This setup validates environment configuration and shows how to build a retrieval + generation pipeline.

## 🚀 Features
- Load text documents with `TextLoader`
- Split text into chunks using `CharacterTextSplitter`
- Generate embeddings via HuggingFace
- Store and query vectors with FAISS
- Use `ChatOpenAI` for conversational responses
- Chain components with LangChain Core

## 🛠️ Tech Stack
- Python
- LangChain Core
- HuggingFace Embeddings
- FAISS
- OpenAI API

## 📂 Usage
1. Install dependencies with pip.
2. Import modules and configure embeddings.
3. Load and split text documents.
4. Store embeddings in FAISS.
5. Query with `ChatOpenAI`.
